# Online Library System

This is a simple Online Library System built using React + Vite.  
The application allows users to browse books, view details, and add new books.

## How to Run the Project

1. Install dependencies:
   npm install

2. Start the development server:
   npm run dev

3. Open the application in the browser:
   http://localhost:5173

## Features
- Browse books by category
- Search books by title or author
- View detailed information of books
- Add new books using Redux
- Responsive UI with Tailwind CSS

## Technologies Used
- React
- Vite
- React Router
- Redux Toolkit
- Tailwind CSS
